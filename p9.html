Project #9: Experimental Camera:

Project #9: Experimental Camera

Date:

5/05/24

Medium:

P5 Code

Decription:

Quick test with the camera, intended to be implemented into a meta game including the player themselves.

Reflections:

It was interesting learning about the ability to use the camera and the interactive possibilities that brings with java script. In the future I would like to experiment more with its capabilities and how the data input from the user can be used in other methods.

Proposal:

I wanted to focus on creating a camera filter that would be used within a game context, involving direct interaction between the player and the character themsevles.
